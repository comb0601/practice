- DirectX 11 SDK (included in Windows SDK)

### Code Standards:
- C++17 standard
- Modern DirectX 11 API
- COM smart pointers where applicable
- Consistent coding style

## Compilation Instructions

### Using Visual Studio:
1. Create Empty C++ Project
2. Add .cpp file
3. Link libraries: d3d11.lib dxgi.lib d3dcompiler.lib user32.lib
4. Build and Run (F5)

### Using Command Line:
```bash
cl.exe /EHsc /std:c++17 Example.cpp /link d3d11.lib dxgi.lib d3dcompiler.lib user32.lib /SUBSYSTEM:WINDOWS
```
