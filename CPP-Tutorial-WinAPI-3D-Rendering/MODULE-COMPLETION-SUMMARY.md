## Compilation

All programs can be compiled with:

**Visual C++ (cl.exe):**
```batch
cl /D "UNICODE" /D "_UNICODE" filename.cpp user32.lib gdi32.lib
```

**g++ (MinGW):**
```batch
g++ -D UNICODE -D _UNICODE filename.cpp -o output.exe -mwindows -lgdi32 -luser32
```
