## üíª How to Use This Course

### Compiling Programs

#### Windows with Visual Studio:
```bash
cl /EHsc /std:c++17 program.cpp
program.exe
```

#### Windows with g++:
```bash
g++ -std=c++17 program.cpp -o program
program.exe
```

#### Linux/Mac:
```bash
g++ -std=c++17 program.cpp -o program
./program
```

#### For DirectX programs (Windows only):
```bash
cl /EHsc /std:c++17 program.cpp d3d11.lib dxgi.lib d3dcompiler.lib
```

## üêõ Troubleshooting

### Common Issues:

**"Cannot open include file"**
- Solution: Install Windows SDK and DirectX SDK

**"Unresolved external symbol"**
- Solution: Link required libraries (d3d11.lib, dxgi.lib, etc.)

**"error C2679: binary '<<' : no operator found"**
- Solution: Include correct headers (#include <iostream>)

**DirectX initialization fails**
- Solution: Update graphics drivers, check DirectX version
