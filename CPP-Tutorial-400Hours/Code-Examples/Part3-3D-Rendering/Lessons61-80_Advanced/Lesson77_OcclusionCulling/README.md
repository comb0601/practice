# Lesson 77: Occlusion Culling

## Overview
Advanced 3D rendering technique demonstration.

## Building
```bash
cmake --build . --target Lesson77_OcclusionCulling
./bin/Lessons61-80_Advanced/Lesson77_OcclusionCulling
```

## Controls
- WASD: Move camera
