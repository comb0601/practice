# Lesson 73: Compute Shader Examples

## Overview
Advanced 3D rendering technique demonstration.

## Building
```bash
cmake --build . --target Lesson73_ComputeShader
./bin/Lessons61-80_Advanced/Lesson73_ComputeShader
```

## Controls
- WASD: Move camera
- Mouse: Look around
- Scroll: Zoom
- ESC: Exit
