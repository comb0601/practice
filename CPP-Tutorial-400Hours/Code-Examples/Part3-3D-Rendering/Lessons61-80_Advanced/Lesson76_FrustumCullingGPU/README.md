# Lesson 76: GPU Frustum Culling

## Overview
Advanced 3D rendering technique demonstration.

## Building
```bash
cmake --build . --target Lesson76_FrustumCullingGPU
./bin/Lessons61-80_Advanced/Lesson76_FrustumCullingGPU
```

## Controls
- WASD: Move camera
- Mouse: Look around
- Scroll: Zoom
- ESC: Exit
