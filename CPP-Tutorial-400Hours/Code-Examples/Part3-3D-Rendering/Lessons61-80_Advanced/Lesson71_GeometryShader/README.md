# Lesson 71: Geometry Shader Examples

## Overview
Advanced 3D rendering technique demonstration.

## Building
```bash
cmake --build . --target Lesson71_GeometryShader
./bin/Lessons61-80_Advanced/Lesson71_GeometryShader
```

## Controls
- WASD: Move camera
- Mouse: Look around
- Scroll: Zoom
- ESC: Exit
