# Lesson 61: Deferred Rendering System

## Overview
Advanced 3D rendering technique demonstration.

## Building
```bash
cmake --build . --target Lesson61_DeferredRendering
./bin/Lessons61-80_Advanced/Lesson61_DeferredRendering
```

## Controls
- WASD: Move camera
- Mouse: Look around
- Scroll: Zoom
- ESC: Exit
