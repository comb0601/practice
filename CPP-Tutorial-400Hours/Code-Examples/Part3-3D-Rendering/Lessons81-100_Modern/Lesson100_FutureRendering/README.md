# Lesson 100: Modern Techniques and Future

## Overview
Advanced 3D rendering technique demonstration.

## Building
```bash
cmake --build . --target Lesson100_FutureRendering
./bin/Lessons81-100_Modern/Lesson100_FutureRendering
```

## Controls
- WASD: Move camera
- Mouse: Look around
- Scroll: Zoom
- ESC: Exit
