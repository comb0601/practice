# Lesson 86: Volumetric Lighting God Rays

## Overview
Advanced 3D rendering technique demonstration.

## Building
```bash
cmake --build . --target Lesson86_VolumetricLighting
./bin/Lessons81-100_Modern/Lesson86_VolumetricLighting
```

## Controls
- WASD: Move camera
- Mouse: Look around
- Scroll: Zoom
- ESC: Exit
