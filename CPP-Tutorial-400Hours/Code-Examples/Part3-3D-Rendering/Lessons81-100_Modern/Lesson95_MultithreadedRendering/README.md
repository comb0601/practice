# Lesson 95: Multi-threaded Rendering

## Overview
Advanced 3D rendering technique demonstration.

## Building
```bash
cmake --build . --target Lesson95_MultithreadedRendering
./bin/Lessons81-100_Modern/Lesson95_MultithreadedRendering
```

## Controls
- WASD: Move camera
- Mouse: Look around
- Scroll: Zoom
- ESC: Exit
