# Lesson 81: Advanced Shader Techniques

## Overview
Advanced 3D rendering technique demonstration.

## Building
```bash
cmake --build . --target Lesson81_AdvancedShaders
./bin/Lessons81-100_Modern/Lesson81_AdvancedShaders
```

## Controls
- WASD: Move camera
- Mouse: Look around
- Scroll: Zoom
- ESC: Exit
