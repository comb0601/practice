# Lesson 82: Procedural Generation

## Overview
Advanced 3D rendering technique demonstration.

## Building
```bash
cmake --build . --target Lesson82_ProceduralTextures
./bin/Lessons81-100_Modern/Lesson82_ProceduralTextures
```

## Controls
- WASD: Move camera
- Mouse: Look around
- Scroll: Zoom
- ESC: Exit
