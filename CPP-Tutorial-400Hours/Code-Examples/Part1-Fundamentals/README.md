# Part 1: Fundamentals - Complete Code Examples

## Overview
This directory contains 80+ complete, executable C++ code examples covering all fundamental C++ concepts from basics to advanced topics.

## Compilation
All files can be compiled with Visual Studio (MSVC) using C++17 standard:

```bash
# Basic compilation
cl Lesson01_HelloWorld.cpp

# With C++17 standard
cl /std:c++17 LessonXX_Name.cpp

# With debugging symbols
cl /Zi LessonXX_Name.cpp
```
