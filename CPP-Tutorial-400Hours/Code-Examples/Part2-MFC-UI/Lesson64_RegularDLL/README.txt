FILES:
- RegularDLLApp.h/cpp: Application class

- Resource.h: Resource IDs

1. Create new MFC Dialog Application in Visual Studio
2. Copy files to project directory

4. Run (F5)

This lesson demonstrates Regular MFC DLL with complete working examples.
