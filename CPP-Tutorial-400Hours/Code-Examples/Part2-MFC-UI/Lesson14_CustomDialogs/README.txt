LESSON 14: CUSTOM DIALOG CLASSES
================================================================================

WHAT THIS DEMONSTRATES:
- Derived dialogs

FILES:
- CustomDialogsApp.h/cpp: Application class
- CustomDialogsDlg.h/cpp: Main dialog class
- Resource.h: Resource IDs
- CustomDialogs.rc: Dialog resource
- StdAfx.h/cpp: Precompiled headers

TO COMPILE:
1. Create new MFC Dialog Application in Visual Studio
2. Copy files to project directory
3. Build Solution (Ctrl+Shift+B)
4. Run (F5)

KEY CONCEPTS:
This lesson demonstrates Custom Dialog Classes with complete working examples.

WHERE TO LOOK:
- CustomDialogsDlg.cpp: OnButtonTest() for main demo
- CustomDialogs.rc: Dialog layout
- Resource.h: Control IDs
