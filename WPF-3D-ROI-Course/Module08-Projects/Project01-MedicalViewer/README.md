# Project 8.1: Medical Image Viewer

## Overview
Build a complete medical image viewer with DICOM support, multi-planar reconstruction (MPR), ROI analysis, and window/level controls.

## Project Goals
- Load and display DICOM image series
- Implement MPR (axial, sagittal, coronal views)
- Create ROI tools for measurement and analysis
- Add window/level adjustment controls
- Display patient and study information
