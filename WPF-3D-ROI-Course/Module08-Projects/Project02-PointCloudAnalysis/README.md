# Project 8.2: Point Cloud Analysis Tool

## Overview
Build a point cloud analysis tool that loads PLY/XYZ/LAS files, applies ROI filtering, and calculates statistics.

## Project Goals
- Load point cloud data (PLY, XYZ, LAS formats)
- Render large point clouds efficiently
- Create ROI selection for point filtering
- Calculate density and distribution statistics
- Export filtered point sets
