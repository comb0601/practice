# Lesson 3.1: Camera Basics

## Overview
Understand the fundamentals of 3D cameras, comparing perspective and orthographic projections.

## Learning Objectives
- Learn perspective camera properties (FOV, near/far planes)
- Understand orthographic camera properties
- Compare perspective vs orthographic rendering
- Control camera position and look direction
