## ğŸš€ Getting Started

### Prerequisites
- Computer with modern OS (Windows, Linux, or macOS)
- 8GB RAM (16GB recommended)
- Graphics card supporting OpenGL 3.3+
- Basic computer literacy
- Enthusiasm to learn!

### Required Software

#### 1. C++ Compiler
**Windows:**
```bash
# Option 1: Visual Studio Community (Recommended)
Download from: https://visualstudio.microsoft.com/

# Option 2: MinGW-w64
Download from: https://sourceforge.net/projects/mingw-w64/
```

**Linux (Ubuntu/Debian):**
```bash
sudo apt update
sudo apt install build-essential g++ gdb cmake
```

**Linux (Fedora):**
```bash
sudo dnf install gcc-c++ gdb cmake
```

**macOS:**
```bash
xcode-select --install
# Or use Homebrew
brew install gcc cmake
```

#### 2. Qt Framework

**Download Qt:**
```bash
# Visit: https://www.qt.io/download
# Install Qt 6.x (or Qt 5.15+ LTS)
# Select components:
#   - Qt Creator (IDE)
#   - Qt 6.x for your platform
#   - Qt Charts (optional)
#   - Qt Network
```

**Linux (Ubuntu):**
```bash
sudo apt install qt6-base-dev qt6-tools-dev qt6-tools-dev-tools
sudo apt install qtcreator
```

#### 3. Additional Tools

**CMake:**
```bash
# Windows: Download from https://cmake.org/
# Linux:
sudo apt install cmake  # Ubuntu/Debian
sudo dnf install cmake  # Fedora

# macOS:
brew install cmake
```

**Git (Optional):**
```bash
# Windows: Download from https://git-scm.com/
# Linux:
sudo apt install git

# macOS:
brew install git
```
